API Documentation

1. GET /getUsers

Fetch all users data.

Request: Method: GET, URL: /getUsers

Response: Status Code: 200, Body: { "message": "Users data fetched successfully", "req_data": [array of users] }

2. POST /signup

Register a new user.

Request: Method: POST, URL: /signup

Response: Status Code: 200, Body: { "message": "User registered successfully", "user": { user object } }

3. POST /login

User login.

Request: Method: POST, URL: /login

Response: Status Code: 200, Body: { "message": "Login successful", "user": { user object } }

4. POST /addfooddata

Upload a new food item with image.

Request: Method: POST, URL: /addfooddata

Response: Status Code: 201, Body: { "message": "Food item added successfully" }

5. GET /getfooddata

Retrieve all food items.

Request: Method: GET, URL: /getfooddata

Response: Status Code: 200, Body: Array of food items.
